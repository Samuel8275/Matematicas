# Algebra

El algebra es una rama de las matematicas en las que se estudian las operaciones aritmeticas, pero añadiendo simbolos que representan incognitas.

## Monomios

### Ejemplos

- 2x = 1º
- 6x^2 = 2º
-  x^100 = 100º
- 14y^3 = 3º
- 4yz = 2º
- 1,4t = 1º

### Sumas

- 4x + 5x = 9x
- 5y + 9y = 14y
- 8xy + 3xy = 11xy
- 3x + 5y = No se puede
- 4xy + 3x = x(4y + 3)

### Restas

- 5y - 2y = 3y
- 4xyz - 8xyz = -4xyz
- t - t = 0
- x^2 - 2x^2 = -x^2
- 60y - 2y = 58y

## Polinomios

### Ejemplos

- 2x + 3y
- 8x - 2x^2 
- 4z + 8z - 7zy
- 8t^5 - t + 3t^4 + 16t^2

### Ordenar

- 2x + 3x^3 = 3x^3 + 0x^2 + 2x
- x^3 - 7x + 2x^2 = x^3 + 2x^2 - 7x 
- -y^3 - y^2 + 4y^4 - 8 = 4y^4 - y^3 - y^2 + 0y - 8
